# 1.__int128
(1) __int128 是 GCC/Clang 编译器扩展提供的一个 128位有符号整数 类型。它能够表示的范围非常大，适合处理超大的整数。
(2) 范围:
+ 有符号:__int128：约 -1.7×10³⁸ 到 1.7×10³⁸
+ 无符号:__uint128_t：约 0 到 3.4×10³⁸
(3) 对比
+ int：通常 32位，约 ±21亿
+ long long：64位，约 ±9.2×10¹⁸
+ __int128：128位，约 ±1.7×10³⁸

# 4.binary_search 是 C++ 标准库中用于在有序区间内进行二分查找的函数
bool found = binary_search(起始迭代器, 结束迭代器, 要查找的值);
⚠️ 注意事项
+ 必须保证区间是有序的，否则结果不确定
+ 如果区间有多个相同的值，找到任意一个就返回 true
+ 只返回是否存在，不返回位置（要获取位置用 lower_bound 或 find）

# 6.范围 for 循环
```
// 范围 for 循环
for (int v : graph[u]) {
    // 处理 v
}

// 等价于传统循环
for (int idx = 0; idx < graph[u].size(); idx++) {
    int v = graph[u][idx];
    // 处理 v
}

// 也等价于迭代器
for (vector<int>::iterator it = graph[u].begin(); it != graph[u].end(); it++) {
    int v = *it;
    // 处理 v
}
```
