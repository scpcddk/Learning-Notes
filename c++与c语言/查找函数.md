# C++ 查找函数完全指南 📚

## 一、二分查找系列（==需要有序区间==）

### 1.1 `binary_search` - 判断元素是否存在
```cpp
#include <algorithm>
bool found = binary_search(开始迭代器, 结束迭代器, 目标值);
```
- **返回值**：`true`（找到）/`false`（没找到）
- **要求**：区间必须有序（升序）
- **时间复杂度**：O(log n)

### 1.2 `lower_bound` - 第一个 ≥ 目标值的位置
```cpp
auto it = lower_bound(开始迭代器, 结束迭代器, 目标值);
```
- **返回值**：指向第一个**大于等于**目标值的迭代器
- **用途**：查找元素首次出现的位置

### 1.3 `upper_bound` - 第一个 > 目标值的位置
```cpp
auto it = upper_bound(开始迭代器, 结束迭代器, 目标值);
```
- **返回值**：指向第一个**大于**目标值的迭代器
- **用途**：确定元素范围的右边界

### 1.4 `equal_range` - 获取相等元素的范围
```cpp
auto [first, last] = equal_range(开始迭代器, 结束迭代器, 目标值);
```
- **返回值**：`pair`，`first` = `lower_bound`，`second` = `upper_bound`
- **用途**：一次性获取所有相等元素的区间
### 1.5 `prev(upper_bound)` - 最后一个小于等于 x 的元素
- **核心原理**：
auto it = upper_bound(v.begin(), v.end(), x);  // 第一个 > x 的元素
if (it != v.begin()) 
{
    --it;  // 现在指向最后一个 <= x 的元素
}
- **可行性**：
+ upper_bound(x) 返回第一个 大于 x 的元素
+ 它前面的那个元素（prev(it)）一定是 小于等于 x 的
+ 而且是最后一个满足条件的

### 📝 二分查找应用示例
```cpp
vector<int> v = {1, 2, 2, 2, 3, 4, 5};

// 1. 检查是否存在
bool exist = binary_search(v.begin(), v.end(), 2);  // true

// 2. 查找第一个2的位置
auto it1 = lower_bound(v.begin(), v.end(), 2);      // 指向索引1

// 3. 查找第一个大于2的位置
auto it2 = upper_bound(v.begin(), v.end(), 2);      // 指向索引4（值为3）

// 4. 统计2的出现次数
int count = upper_bound(v.begin(), v.end(), 2) - lower_bound(v.begin(), v.end(), 2);  // 3

// 5. 获取相等范围
auto range = equal_range(v.begin(), v.end(), 2);    // [it1, it2)
```

## 二、通用查找系列（不要求有序）

### 2.1 `find` 系列
| 函数 | 功能 | 返回值 |
|------|------|--------|
| `find` | 查找第一个等于给定值的元素 | 迭代器 |
| `find_if` | 查找第一个满足条件的元素 | 迭代器 |
| `find_if_not` | 查找第一个不满足条件的元素 | 迭代器 |

```cpp
vector<int> v = {1, 3, 5, 7, 9, 5, 3};

auto it1 = find(v.begin(), v.end(), 5);                    // 找到第一个5
auto it2 = find_if(v.begin(), v.end(), [](int x){          // 第一个偶数（不存在）
    return x % 2 == 0; 
});
auto it3 = find_if_not(v.begin(), v.end(), [](int x){      // 第一个不小于3的数
    return x < 3;
});
```

### 2.2 特殊查找
| 函数 | 功能 | 返回值 |
|------|------|--------|
| `find_end` | 查找最后一个**子序列** | 迭代器 |
| `find_first_of` | 查找任一指定元素首次出现 | 迭代器 |
| `adjacent_find` | 查找第一对相邻相等元素 | 迭代器 |

```cpp
vector<int> v = {1, 2, 3, 1, 2, 3, 4, 5};
vector<int> pattern = {2, 3};

// 查找最后一个 {2,3}
auto it1 = find_end(v.begin(), v.end(), pattern.begin(), pattern.end());

// 查找第一个出现在 pattern 中的元素
auto it2 = find_first_of(v.begin(), v.end(), pattern.begin(), pattern.end());

// 查找相邻相等的元素
auto it3 = adjacent_find(v.begin(), v.end());
```

## 三、计数系列

| 函数 | 功能 | 返回值 |
|------|------|--------|
| `count` | 统计等于给定值的元素个数 | `size_t` |
| `count_if` | 统计满足条件的元素个数 | `size_t` |

```cpp
vector<int> v = {1, 2, 2, 2, 3, 4, 2};

int c1 = count(v.begin(), v.end(), 2);                    // 4
int c2 = count_if(v.begin(), v.end(), [](int x){          // 偶数个数：5
    return x % 2 == 0;
});
```

## 四、最大/最小值系列

| 函数 | 功能 | 返回值 |
|------|------|--------|
| `min_element` | 返回最小元素的迭代器 | 迭代器 |
| `max_element` | 返回最大元素的迭代器 | 迭代器 |
| `minmax_element` | 同时返回最小和最大元素的迭代器 | `pair` |

```cpp
vector<int> v = {3, 1, 4, 1, 5, 9, 2};

auto min_it = min_element(v.begin(), v.end());            // 指向第一个 1
auto max_it = max_element(v.begin(), v.end());            // 指向 9
auto [min_it2, max_it2] = minmax_element(v.begin(), v.end());  // 同时获取
```

## 五、子序列搜索系列

| 函数 | 功能 | 返回值 |
|------|------|--------|
| `search` | 查找第一个子序列 | 迭代器 |
| `search_n` | 查找连续 n 个等于给定值的元素 | 迭代器 |

```cpp
vector<int> v = {1, 2, 3, 4, 1, 2, 3, 4, 5};
vector<int> pattern = {2, 3, 4};

// 查找第一个 {2,3,4}
auto it1 = search(v.begin(), v.end(), pattern.begin(), pattern.end());

// 查找连续2个4
auto it2 = search_n(v.begin(), v.end(), 2, 4);  // 不存在，返回 end()
```

## 六、==实用技巧速查表==

| 需求 | 有序数组 | 无序数组 |
|------|---------|---------|
| 检查元素是否存在 | `binary_search` | `find != end` |
| 获取元素位置 | `lower_bound` | `find` |
| 统计出现次数 | `upper_bound - lower_bound` | `count` |
| 第一个大于 x 的元素 | `upper_bound` | `find_if(>x)` |
| 最后一个小于等于 x 的元素 | `prev(upper_bound)` | 需要遍历 |
| 所有等于 x 的范围 | `equal_range` | 需要排序 |

## 七、完整示例：综合应用

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    vector<int> v = {1, 2, 2, 2, 3, 4, 5, 5, 6, 8, 8, 9};
    
    // 确保有序（已有序）
    
    // 1. 查找元素
    int x = 2;
    if (binary_search(v.begin(), v.end(), x)) {
        cout << x << " 存在" << endl;
    }
    
    // 2. 获取位置
    auto it = lower_bound(v.begin(), v.end(), x);
    int pos = it - v.begin();
    cout << "第一个 " << x << " 在位置 " << pos << endl;
    
    // 3. 统计出现次数
    int cnt = upper_bound(v.begin(), v.end(), x) - lower_bound(v.begin(), v.end(), x);
    cout << x << " 出现了 " << cnt << " 次" << endl;
    
    // 4. 查找范围
    auto [first, last] = equal_range(v.begin(), v.end(), x);
    cout << "范围: [" << first - v.begin() << ", " << last - v.begin() << ")" << endl;
    
    // 5. 查找插入位置
    int insert_val = 7;
    auto insert_pos = lower_bound(v.begin(), v.end(), insert_val);
    cout << insert_val << " 应该插入在位置 " << insert_pos - v.begin() << endl;
    
    // 6. 查找边界
    int limit = 5;
    auto first_greater = upper_bound(v.begin(), v.end(), limit);
    cout << "第一个大于 " << limit << " 的元素是 " << *first_greater << endl;
    
    auto last_less_or_equal = prev(upper_bound(v.begin(), v.end(), limit));
    cout << "最后一个小于等于 " << limit << " 的元素是 " << *last_less_or_equal << endl;
    
    return 0;
}
```

## ==总结== ✨
- **有序数组**：优先使用二分查找系列（O(log n)）
- **无序数组**：使用通用查找系列（O(n)）
- **需要位置**：用 `lower_bound`/`find` 而不是 `binary_search`
- **需要计数**：有序用差值，无序用 `count`
- **需要范围**：`equal_range` 最方便
