# 位运算
## 注意
1.10进制整型和字符型数据参与位运算时，用其补码进行运算
2.八进制与十六进制整数参与位运算时，将其直接转成二进制运算
## 一.& 按位与 （AND）
和0做与运算为清零运算，和1做与运算保持不变
## 二.| 按位或 （OR）
和1做或运算为置1操作，和0做或运算保持不变
## 三.^ 按位异或 （XOR）
1.相同为0，不同为1
2.与0做异或运算结果不变，与1做异或运算结果取反
3.两个相同数异或运算结果为0
+ 例如交换两个整型变量的值
char a=3,b=4;
a=a^b;//a=3^4=7
b=b^a;//b=4^7=3
a=a^b;//a=7^3=4;
## 四.<< 左移    >> 右移
1. 无符号数左移（ << ）
- 规则：左移时，右边补0，高位直接移出（溢出舍弃）。
- 效果：左移 n 位，相当于乘以 2^n （前提是没有溢出）。
2. 无符号数右移（ >> ）
- 规则：右移时，左边补0，低位直接移出。
- 效果：右移 n 位，相当于除以 2^n （向下取整）。
## 五.~ 按位求反 （一元运算符）
1.符号位也取反
2.按位求反的结果和原数的关系为~x = -x -1
## 六.运用
1. 小写字母转大写：ch = ch & (~32)
2. 大写字母转小写：ch = ch | 32
3.大小写字母互相转换：ch = ch ^ 32
