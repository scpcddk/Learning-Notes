# ==笔记==

## 1.文件删除
+ 能否删除一个文件，不取决于这个文件本身的权限，而**取决于你对文件所在的父目录有没有“写”权限**
+ 父目录的 w 权限 = 允许修改这个目录的内容（即：增、删、重命名里面的文件）
+ 文件自己的 w 权限 = 允许修改这个文件的内容
+ **删除文件 = 修改父目录的内容**（把文件名从目录列表中移除），不需要对文件本身有写权限。

## 2.usermod vs useradd

| 命令 | 作用 | 适用场景 |
|------|------|----------|
| `useradd` | 创建新用户 | 用户还不存在时 |
| `usermod` | 修改已存在的用户 | 用户已经存在时 |

+ -a = append（追加，不覆盖原有组）
+ -G = 附加组列表
+ -aG = 加组不覆盖

## 3.su - root 切换用户；sudo 是提权执行命令

| 命令 | 作用 | 是否切换用户 | 环境变量 |
|------|------|--------------|----------|
| `sudo command` | 以 root 身份执行单条命令 | 否（还是你自己） | 保留原用户环境 |
| `su -` | 完全切换成 root 用户 | 是 | 加载 root 的环境变量 |
