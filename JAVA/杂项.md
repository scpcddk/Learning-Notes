# 1.final修饰的变量里面的内容无法发生改变
+ final 基本类型 vs 引用类型：
基本类型：数值本身不可变。
引用类型：引用地址不可变（即不能再指向其他对象），但对象内部的状态是可以改变的。
# 2.枚举
-   每一个枚举项，都是该枚举类的对象
-   枚举项在底层其实就是常量，默认用 `public static final` 修饰
-   枚举类的第一行上必须是枚举项，枚举项之间用逗号隔开，以分号作为结尾
-   枚举类的构造方法必须是 `private` 修饰，不让外界创建本类的对象
-   编译器会给枚举类新增两个默认存在的方法：`values()`、`valueOf()`
# 3.方法重写注意事项和要求
+ 重写方法的名称、形参列表必须与父类中的一致，方法体按照实际需求书写。
+ **了解**：子类重写父类方法时，访问权限子类必须大于等于父类（空着不写 < protected < public）
+ **了解**：子类重写父类方法时，返回值类型子类必须小于等于父类
+ **建议**：重写的方法申明和父类保持一致即可
+ final修饰类为最终类，里面所有的方法不能被重写
+ private私有方法、static静态方法、final最终方法不能被重写
