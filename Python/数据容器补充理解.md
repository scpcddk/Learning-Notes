# 列表

## 1.去重

+ 需求：去掉列表中所有等于最大值和最小值的元素，求剩余数据的中位数。
+ 两种常见错误：
  + remove(max) + remove(min)	（只删第一个匹配项，重复极值残留）
  + 排序后切片 [1:-1]	（只掐头去尾各1个，重复极值同样残留）
+ 正确做法：去极值要用“过滤思维”（剔除所有等于极值的元素），而不能用“删除思维”

```python
model_id, _, _ = params
```

# 字典

## 1.get() 方法可以指定默认值。统计计数的经典写法是：

```python
counts["resnet50"] = counts.get("resnet50", 0) + 1
```

这句话的逻辑是：从字典取 "resnet50" 的值，如果不存在就返回 0，然后加 1，再存回去。这样就安全地完成了累加，不会因为键不存在而报错

## 2.Python 的 collections 模块里有一个 Counter 类，专门用来统计频次。它可以直接接收一个列表，自动生成 {元素: 出现次数} 的字典，还提供了 .most_common(1) 方法直接返回出现次数最多的元素。

# 集合

## 1.时间复杂度
+ **集合（哈希表）**：查找时间复杂度 O(1) —— 直接哈希计算位置。
+ **列表按值查找**：时间复杂度 O(n) —— 只能从头到尾逐个比较。
+ **列表按索引查找**：时间复杂度 O(1) —— 直接计算内存地址。
