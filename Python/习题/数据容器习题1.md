### 四、简答题

1. 元组是不可变类型，其哈希值固定不变，因此可作为字典的键；列表是可变类型，无法计算稳定哈希值，所以不能作为键。
2. `list.copy()`（或切片`[:]`）是浅拷贝，创建了一个新的列表对象，但内部元素仍然引用原对象（如果元素是可变对象则共享）；直接赋值只是让新变量指向同一个列表对象，修改会相互影响。
3. `{char: s.count(char) for char in set(s)}` 或者用 `collections.Counter(s)` 转换成字典，如 `dict(collections.Counter(s))`。最简单：使用字典推导 `{c: s.count(c) for c in s}` 也可。

### 五、编程题

1. 示例代码（使用字典保持顺序）
```python
lst = [4, 2, 2, 8, 4, 1, 8]
new_lst = list(dict.fromkeys(lst))
print(new_lst)   # [4, 2, 8, 1]
```

2. 示例代码

```python
text = "apple banana apple orange banana apple"
words = text.split()
freq = {}
## for w in words:
freq[w] = freq.get(w, 0) + 1
print(freq)   # {'apple': 3, 'banana': 2, 'orange': 1}
```

也可用字典推导 + `count`，但效率较低；或用 `collections.Counter`。

# 题后回顾

## 1.填空题第5题
{x: x*2 for x in 'AB'} 遍历字符串 'AB' 中的两个字符 'A' 和 'B'，
键是字符本身，值是字符重复两次：

```Python
'A' * 2 → 'AA'
'B' * 2 → 'BB'
```

所以结果是 {'A': 'AA', 'B': 'BB'}。

+ 注意：x 是字符串，* 是重复，不是数学乘法。如果是数字，2*2 才是 4。

## 2.简答题第2题

### (1)字典推导式统计字符

```python
{ch: s.count(ch) for ch in set(s)}
```

+ set(s) 去重得所有字符
+ 遍历字符，用 s.count(ch) 算次数
+ 组成字典

### (2)用 Counter 统计

```python

from collections import Counter  
d = dict(Counter(s))
```

+ Counter 是自动计数器
+ 丢给它字符串，直接统计好
+ 套上 dict() 变成普通字典

## 3.编程题第2题
统计单词出现次数（循环字典法）

```python
freq = {}
for word in words:
    freq[word] = freq.get(word, 0) + 1
```

+ get(word, 0) 安全取值，没有就返回0
+ 每次 +1 存回去
+ 结果就是单词 -> 次数的字典
