# Python核心语法

## 一. 动态类型特性
- 变量即标签（引用）
- 类型随赋值改变
- 一切皆对象

## 二. 核心数据结构
- 列表（可变序列）
- 元组（不可变序列）
- 字典（哈希映射）
- 集合（无序不重复）
- 切片操作

## 三. 函数特色
- 多返回值（自动打包为元组）
- 默认参数可变对象陷阱
- 关键字参数与 `*args` / `**kwargs`
- lambda 匿名函数
- 装饰器

## 四. 迭代与生成
- 列表推导式
- 生成器表达式
- `yield` 与生成器函数
- `enumerate` / `zip`

## 五. 上下文管理
- `with` 语句
- 自定义上下文管理器（`__enter__` / `__exit__`）

## 六. 属性与封装
- `@property`（替代 getter/setter）
- 命名约定：`_`（保护）与 `__`（私有名称修饰）

## 七. 模块与包
- 每个 `.py` 即模块
- `__name__ == "__main__"`
- `__init__.py` 包结构

## 八. 异常处理
- `try / except / else / finally`
- `raise`

## 九. 常用内置函数
- `len` / `range` / `enumerate` / `zip`
- `map` / `filter` / `sorted`
- `any` / `all`

## 十. 开发工具链
- pip 包管理
- 虚拟环境（venv）
- 类型注解（typing 模块，可选）
