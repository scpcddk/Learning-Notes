# Python核心语法(对比化学习)(凡未标注内容皆为差别微小)

## **一、入门基础**

### **(一).运行原理**

#### 1.运行方式

+ 交互式（REPL）：终端输入 `python`，逐行执行，即时反馈
+ 脚本式：编写 `.py` 文件，执行 `python filename.py`

#### 2.执行原理

+ 源码(.py) → 字节码(.pyc) → Python虚拟机(PVM) → 运行结果
+ Python 是==解释型语言==（严格说是"先编译后解释"）
+ 字节码是平台无关的中间代码

---

### **(二).动态类型特性**（变量即标签，类型随赋值改变）

+ **变量是标签，不是盒子**
+ **变量本身没有类型，对象才有类型**,无需声明类型，运行时动态确定
+ 赋值只是让变量"指向"对象,同一变量可先后指向**不同类型**的对象
+ 多个变量可以指向同一个对象：a = b = [1, 2, 3]

```python
a = 10      # a 标签贴在 10 这个对象上
a = "hello" # a 标签撕下来，贴到 "hello" 对象上
```

---

### **(三).==一切皆对象==**

#### 1.什么是"对象"

+ 对象 = 数据（属性）+ 行为（方法）
+ 对象存储在堆内存中，由垃圾回收器管理

#### 2.所有 Python 元素都是对象

+ 可以动态给对象添加属性
+ 可以将函数作为参数传递（高阶函数）
+ 所有类型都有统一的类型体系

### ==**(一).字面量与变量**==

#### **1.字面量与c++对比**

+ 字面量是直接书写的固定值
+ type()语句可得到数据类型，`type(要查看的数据类型)`
+ isinstance()检查是否属于指定的类型，返回bool值，`isinstance(数据，类型)`

| 类别 | Python | C++ | 说明 |
| ------ | -------- | ----- | ------ |
| 整数精度 | **无限精度，无溢出** | 受类型范围限制 | Python可计算超大整数 |
| 数字分隔符 | 支持 `1_000_000` | 不支持 | 提高可读性 |
| 布尔值 | `True` / `False`（首字母大写） | `true` / `false` | Python中布尔是int子类 |
| 布尔运算 | `True + True == 2` | 不允许 | **Python布尔可参与算术** |
| 多行字符串 | **三引号** `"""..."""` | 需用 `\n` 或宏 | Python原生支持 |
| 原始字符串 | `r"C:\path"` | 无 | **原始字符串(r)**转义字符失效，按原样输出 |

| 对比项 | Python | C++ |

| 指针/引用 | **所有变量都是引用语义** | 分值语义、指针、引用 |

### **(一).缩进规则（无大括号）**

#### 1.缩进（Python vs C++ 核心差异）

| 对比项 | Python | C++ |
| -------- | -------- | ----- |
| 代码块标记 | **缩进**（强制） | 大括号 `{}`（缩进仅用于可读性） |
| 缩进量 | 统一（通常4空格） | 无要求 |
| 混合缩进 | ❌ 禁止（Tab/空格混用报错） | ✅ 允许 |
| 代码块结束 | 缩进减少即结束 | 右大括号 `}` |

#### 2. 核心原理

> [!IMPORTANT]
> ==Python：缩进即语法==

+ 同一代码块必须缩进一致
+ 嵌套层级增加缩进
+ 建议使用4空格（PEP 8标准）

```python
# 缩进决定代码归属
if True:
    print("属于if")      # 4空格缩进
    print("也属于if")
print("不属于if")         # 缩进恢复，if块结束
```

### **(三).match-case**（模式匹配，Python 3.10+）

| 对比项 | Python `match-case` | C++ `switch` |
| -------- | --------------------- | -------------- |
| 语法 | `match 值:`<br>`case 模式:`<br>`代码`<br>`case _:`<br>`默认代码` | `switch (表达式) {`<br>`case 常量:`<br>`代码; break;`<br>`default:`<br>`默认代码; break;`<br>`}` |
| 匹配值类型 | **任意类型** | 整型、枚举、字符 |
| 字符串匹配 | ✅ 支持 | ❌ 不支持 |
| 类型匹配 | ✅ 支持 | ❌ 不支持 |
| 解包匹配 | ✅ 支持（列表、元组） | ❌ 不支持 |
| 条件守卫（guard） | ✅ 支持 | ❌ 不支持 |
| **多个值匹配** | `\|` 分隔 | 连续写多个 `case` |
| `break` 需要 | ❌ **不需要（不会贯穿）** | ✅ 需要 |
| 默认分支 | `case _:`(通配符，匹配任意值) | `default:` |

+ `case x:` 会将匹配到的值**绑定到变量 `x`**，后续代码可以使用该变量。

```python
value = 42
# 使用 case _（不绑定）
match value:
    case 1:
        print("匹配到1")
    case _:
        print("匹配到其他值")   # 无法知道具体是什么值

# 使用 case x（变量绑定）    变量名可以任意取
match value:
    case 1:
        print("匹配到1")
    case other:
        print(f"匹配到: {other}")   # 可以使用 other，输出 42
```

#### 2.列表（list）操作

##### (1).增删改查

| 操作 | 方法/语法 | 说明 | 示例 |
|------|----------|------|------|
| **增** | `lst.append(x)` | 末尾添加 | `[1,2] → [1,2,x]` |
| | `lst.insert(i, x)` | 指定位置插入 | `[1,3] → insert(1,2) → [1,2,3]` |
| | `lst.extend(iter)` | 合并列表 | `[1,2] → extend([3,4]) → [1,2,3,4]` |
| **删** | `lst.pop()` | 移除并返回最后一个 | `[1,2,3] → pop() → 3, lst=[1,2]` |
| | `lst.pop(i)` | 移除并返回**索引** i | `[1,2,3] → pop(1) → 2, lst=[1,3]` |
| | `lst.remove(x)` | 移除第一个值为 x 的元素 | `[1,2,1] → remove(1) → [2,1]` |
| | `lst.clear()` | 清空所有元素 | `[1,2,3] → clear() → []` |
| | `del lst[i]` | 删除索引 i 的元素 | `[1,2,3] → del lst[1] → [1,3]` |
| **改** | `lst[i] = x` | 直接赋值修改 | `[1,2] → lst[1]=10 → [1,10]` |
| **查** | `lst[i]` | 访问索引 i 的元素 | `[1,2,3][1] → 2` |
| | `lst.index(x)` | 返回第一个值为 x 的索引 | `[1,2,3].index(2) → 1` |
| | `x in lst` | 判断是否存在 | `2 in [1,2,3] → True` |
| | `lst.count(x)` | 统计 x 出现次数 | `[1,2,1].count(1) → 2` |

##### (2).排序与反转

| 操作 | 方法/函数 | 说明 | 示例 |
|------|----------|------|------|
| 排序 | `lst.sort()` | 原地升序排序 | `[3,1,2] → sort() → [1,2,3]` |
| 降序 | `lst.sort(reverse=True)` | 原地降序排序 | `[1,2,3] → sort(reverse=True) → [3,2,1]` |
| 反转 | `lst.reverse()` | 原地反转 | `[1,2,3] → reverse() → [3,2,1]` |
| 排序（新） | `sorted(lst)` | 返回新列表，原列表不变 | `sorted([3,1,2]) → [1,2,3]` |

##### (3).复制

| 方法 | 说明 |
|------|------|
| `lst2 = lst.copy()` | 浅拷贝 |
| `lst2 = lst[:]` | 切片复制（浅拷贝） |
| `copy.deepcopy(lst)` | 深拷贝（需 `import copy`） |

##### (4).解包操作

| 操作 | 语法 | 示例 |
|------|------|------|
| 基础解包 | `a, b, c = [1,2,3]` | `a=1, b=2, c=3` |
| 变量交换 | `x, y = y, x` | `x=10, y=20 → x=20, y=10` |
| 星号解包 | `first, *rest = [1,2,3,4]` | `first=1, rest=[2,3,4]` |
| 星号解包（尾） | `*first, last = [1,2,3,4]` | `first=[1,2,3], last=4` |

##### (5).列表推导式

| 类型 | 语法 | 示例 |
|------|------|------|
| 基础 | `[表达式 for 变量 in 可迭代对象]` | `[x**2 for x in range(5)] → [0,1,4,9,16]` |
| 带条件 | `[表达式 for 变量 in 可迭代对象 if 条件]` | `[x for x in range(10) if x%2==0] → [0,2,4,6,8]` |
| 带 if-else | `[真值 if 条件 else 假值 for 变量 in 可迭代对象]` | `["偶" if x%2==0 else "奇" for x in range(3)] → ["偶","奇","偶"]` |
| 嵌套 | `[表达式 for 变量1 in 可迭代1 for 变量2 in 可迭代2]` | `[(x,y) for x in range(2) for y in range(2)] → [(0,0),(0,1),(1,0),(1,1)]` |

##### (6).其他常用操作

| 操作 | 语法 | 说明 | 示例 |
|------|------|------|------|
| 拼接 | `lst1 + lst2` | 合并两个列表，返回新列表 | `[1,2] + [3,4] → [1,2,3,4]` |
| 重复 | `lst * n` | 列表重复 n 次，返回新列表 | `[1,2] * 3 → [1,2,1,2,1,2]` |
| 长度 | `len(lst)` | 返回列表元素个数 | `len([1,2,3]) → 3` |
| 遍历元素 | `for item in lst:` | 遍历每个元素 | 输出 `1,2,3` |
| 遍历索引+元素 | `for i, item in enumerate(lst):` | 同时获取索引和值 | `0:1, 1:2, 2:3` |

### **(二).元组（不可变，组包解包，多返回值的本质）**

> [!IMPORTANT]
> **元组与列表最大区别是不可变**

| 对比项 | 元组（tuple） | 列表（list） |
|--------|--------------|--------------|
| 可变性 | ❌ 不可变 | ✅ 可变 |
| 语法 | `(1, 2, 3)` | `[1, 2, 3]` |
| 速度 | 更快 | 较慢 |
| 内存 | 更小 | 较大 |
| 可哈希 | ✅ 是（可作为字典键） | ❌ 否 |
| 使用场景 | 常量数据、函数返回值 | 动态数据、频繁增删改 |

#### 1.tuple对比

| 对比项 | Python `tuple` | C++ `std::tuple` |
|--------|----------------|-------------------|
| 可变性 | **❌ 不可变** | ❌ 不可变 |
| 元素类型 | 任意类型混合 | 编译时确定 |
| 语法 | `(1, "a", 3.14)` | `std::tuple<int, string, double>` |
| 解包 | ✅ 原生支持 | ✅ `std::get` / 结构化绑定 (C++17) |

#### 2.创建元组

```python
# 基本创建
t1 = (1, 2, 3)
t2 = 1, 2, 3           # 括号可省略
t3 = (1,)              # 单元素元组（逗号必须）
t4 = ()                # 空元组
t5 = tuple([1, 2, 3])  # 从列表转换
t6 = tuple("abc")      # ('a', 'b', 'c')

# 混合类型
t = (1, "hello", 3.14, True)
```

#### 3.元组（tuple）常用操作

| 操作 | 代码 | 结果 |
|------|------|------|
| 长度 | `len(t)` | `4` |
| 统计次数 | `t.count(2)` | `2` |
| 查找索引 | `t.index(2)` | `1` |
| 判断存在 | `2 in t` | `True` |
| 拼接 | `(1,2) + (3,4)` | `(1,2,3,4)` |
| 重复 | `(1,2) * 3` | `(1,2,1,2,1,2)` |
| 遍历元素 | `for item in t: print(item)` | `1 2 2 3` |
| **遍历索引+元素** | `for i,item in enumerate(t): print(f"{i}:{item}")` | `0:1 1:2 2:2 3:3` |
| 更新 | namedtuple._replace() | 返回一个新元组，只更新指定字段的值，其他字段保持不变——兼顾了“不可变性”和“部分可变”的需求 |

+ enumerate是 Python 内置函数，用于在遍历可迭代对象时同时获取索引和值
+ 支持负向索引
+ 支持切片

#### 4.组包与解包

**组包**：多个值自动打包成元组。  
**解包**：将元组中的元素分别赋值给多个变量。

> [!NOTE]
> 列表与元组的组包、解包区别：元组默认组包成**元组**，列表需要显式使用 `[]`

| 操作 | 代码 | 结果 |
|------|------|------|
| 组包 | `t = 1, 2, 3` | `(1, 2, 3)` |
| 解包 | `a, b, c = (1, 2, 3)` | `a=1, b=2, c=3` |
| 变量交换 | `x, y = y, x` | 交换 x 和 y |
| 剩余收集 | `a, *b = (1, 2, 3)` | `a=1, b=[2,3]` |
| 函数返回 | `return a, b` | 返回元组 |
| 参数解包 | `func(*tup)` | 元组分拆为参数 |

+ 在元素解包时，`*`表示收集剩余所有元素，允许处理不定数量的元素

### **(三).集合（无序去重，并集/交集/差集）**

#### 1.set对比

| 对比项 | Python `set` | C++ `std::unordered_set` |
|--------|--------------|--------------------------|
| 可变性 | ✅ 可变 | ✅ 可变 |
| 有序性 | ❌ 无序(无法通过索引访问) | ❌ 无序 |
| 元素类型 | 任意可哈希类型 | 同类型 |
| 去重 | ✅ 自动 | ✅ 自动 |

#### 2.创建集合

+ 空集合（注意：**{} 是空字典**）
`s = set()`
`s = {1, 2, 3}`

+ 从列表去重
`lst = [1, 2, 2, 3, 3, 3]`
`s = set(lst)           # {1, 2, 3}`

+ 从字符串

+ 列表推导式创建集合
`s = {x**2 for x in range(5)}  # {0, 1, 4, 9, 16}`

#### 3.常用操作

##### (1).添加与删除

| 操作 | 方法 | 示例 | 说明 |
|------|------|------|------|
| 添加 | `s.add(x)` | `s.add(4)` | 添加单个元素 |
| 添加多个 | `s.update(iter)` | `s.update([5,6])` | 添加多个元素 |
| 删除 | `s.remove(x)` | `s.remove(3)` | 移除元素，不存在报错 |
| 删除安全 | `s.discard(x)` | `s.discard(10)` | 移除元素，不存在不报错 |
| 随机删除 | `s.pop()` | `s.pop()` | **随机移除并返回**一个元素 |
| 清空 | `s.clear()` | `s.clear()` | 清空所有元素 |

##### (2). 其他操作

| 操作 | 代码 | 结果 |
|------|------|------|
| 长度 | `len(s)` | `5` |
| 判断存在 | `3 in s` | `True` |
| 判断不存在 | `10 in s` | `False` |
| 遍历 | `for item in s: print(item)` | **顺序不确定** |
| 浅拷贝 | `s2 = s.copy()` | 复制集合 |
| 浅拷贝 | `s2 = set(s)` | 同上 |
| 不可变集合 | `frozenset([1,2,3])` | 可作为字典键 |

##### (3).**集合运算**

`a = {1, 2, 3, 4}`
`b = {3, 4, 5, 6}`

| 差集 | `a - b` | `a.difference(b)` | `{1, 2}` |
| 对称差集 | `a ^ b` | `a.symmetric_difference(b)` | `{1, 2, 5, 6}` |

##### (4).列表去重

| 方式 | 代码 | 结果 | 是否保持顺序 |
|------|------|------|-------------|
| 字典去重 | `list(dict.fromkeys(lst))` | `[3, 1, 2, 4]` | ✅ 是 |
| 集合去重 | `list(set(lst))` | `[1, 2, 3, 4]` | ❌ 否 |
| 传统循环 | 手动判断追加 | `[3, 1, 2, 4]` | ✅ 是 |

#### 4.集合推导式

| 类型 | 代码 | 结果 |
| ------ | ------ | ------ |
| 基本用法 | `{x**2 for x in range(5)}` | `{0, 1, 4, 9, 16}` |
| 带条件 | `{x for x in range(10) if x % 2 == 0}` | `{0, 2, 4, 6, 8}` |

### **(四).字典**

#### 1

| 对比项 | Python `dict` | C++ `std::unordered_map` |
| -------- | --------------- | -------------------------- |
| 键类型 | **任意可哈希类型（不可变）** | 同类型 |
| 值类型 | **任意类型** | 同类型 |
| 有序性 | Python 3.7+ **插入有序** | 无序 |
| 语法 | `{key: value}` | `{{key, value}}` |
| 访问不存在的键 | 抛出 `KeyError` | 插入默认值（`operator[]`） |

#### 2.创建字典

```python
# 空字典
字典名称 = {}
字典名称 = dict()

# 直接创建
字典名称 = {key: value, key: value···}
字典名称 = {"name": "Alice", "age": 25, "city": "Beijing"}

# dict() 构造
字典名称 = dict(name="Alice", age=25)   # 键自动为字符串
字典名称 = dict([("name", "Alice"), ("age", 25)])  # 从键值对列表

# 字典推导式
squares = {x: x**2 for x in range(5)}  # {0:0, 1:1, 2:4, 3:9, 4:16}
```

#### 3.字典（dict）常用操作

##### (1)**增删改查**

| 分类 | 操作 | 方法/语法 | 说明 | 示例 |
|------|------|----------|------|------|
| **增** | 添加/修改 | `d[key] = value` | 键不存在则添加 | `d["city"] = "Beijing"` → `{"city":"Beijing"}` |
| | 批量更新 | `d.update(other)` | 批量添加或更新 | `d.update({"age":26})` → `{"age":26}` |
| **删** | 删除键 | `del d[key]` | 删除指定键，不存在则 KeyError | `del d["city"]` |
| | 删除并返回值 | `d.pop(key)` | 删除并返回值 | `value = d.pop("age")` |
| | 删除最后一项 | `d.popitem()` | 删除并返回最后插入的键值对 | `k, v = d.popitem()` |
| | 清空 | `d.clear()` | 清空所有键值对 | `d.clear()` |
| **改** | 修改已存在的键 | `d[key] = value` | 键必须已存在（与添加语法相同） | `d["age"] = 26`（假设 age 已存在） |
| **查** | 直接访问 | `d[key]` | 键不存在→KeyError | `print(d["name"])` |
| | 安全访问 | `d.get(key)` | 不存在返回 None | `d.get("name")` |
| | 带默认值 | `d.get(key, default)` | 不存在返回默认值 | `d.get("salary", 0)` |
| | 判断键存在 | `key in d` | 返回 True/False | `"name" in d` |

##### (2)常用方法

| 方法 | 说明 | 示例 |
| ------ | ------ | ------ |
| `d.keys()` | 返回所有键的视图 | `for k in d.keys():` |
| `d.values()` | 返回所有值的视图 | `for v in d.values():` |
| `d.items()` | 返回所有键值对的视图 | `for k, v in d.items():` |
| `d.get(k, default)` | 安全取值，可指定默认值 | `d.get("x", 0)` |
| `d.setdefault(k, default)` | 键不存在则设置默认值并返回 | `d.setdefault("count", 0)` |
| `d.update(other)` | 合并另一个字典 | `d.update({"a":1})` |
| `d.pop(k, default)` | 删除并返回值，可指定默认 | `d.pop("x", None)` |
| `d.copy()` | 浅拷贝 | `d2 = d.copy()` |

+ 视图：
  + **动态**：视图会随原字典变化而变化
  + **可迭代**：可直接用于 `for` 循环
  + **支持成员测试**：`key in d.keys()` 等价于 `key in d`
  + **可转换为列表**：`list(d.keys())` 获取键列表

##### (3)遍历字典

| 遍历方式 | 代码 | 说明 |
| ---------- | ------ | ------ |
| 遍历键 | `for key in d:` | 直接遍历字典，默认识别键 |
| 遍历值 | `for value in d.values():` | 遍历所有值 |
| 遍历键值对 | `for key, value in d.items():` | 同时获取键和值（推荐） |

#### 4.字典合并

```Python
d1 = {"a": 1, "b": 2}
d2 = {"c": 3, "d": 4}
```

+ 方法1：使用 | 运算符（Python 3.9+）
`merged = d1 | d2`          # 新建字典
`d1 |= d2`                 # 原地更新

+ 方法2：解包（Python 3.5+）
`merged = {**d1, **d2}`

+ 方法3：update
`d1.update(d2)`

#### 5.字典推导式

| 类型 | 代码 | 结果/说明 |
| ------ | ------ | ---------- |
| 基础 | `{name: len(name) for name in ["Alice", "Bob", "Charlie"]}` | `{'Alice':5, 'Bob':3, 'Charlie':7}` |
| 带条件 | `{x: x**2 for x in range(10) if x % 2 == 0}` | `{0:0, 2:4, 4:16, 6:36, 8:64}` |
| 反转键值 | `{v: k for k, v in original.items()}` | 值重复会被覆盖（如 `{'a':1, 'b':2, 'c':1}` → `{1:'c', 2:'b'}`） |

#### 6.字典注意事项

+ **键必须可哈希**  
  + ✅ 元组、字符串、数字等不可变类型  
  + ❌ 列表、字典等可变类型  
+ **Python 3.7+ 字典保持插入顺序**  
  + 遍历时按键添加的顺序输出  
+ **遍历时不要修改字典大小**  
  + 错误：`for k in d: del d[k]`  
  + 正确：遍历键的副本 `for k in list(d.keys()):`  
+ **浅拷贝 vs 深拷贝**  
  + `copy()` 只复制一层，嵌套可变对象会被共享  
  + 需要完全独立时用 `copy.deepcopy()`
