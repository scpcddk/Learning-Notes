## **一、入门基础**

### **(一).运行原理**

#### 1.运行方式
+ 交互式（REPL）：终端输入 `python`，逐行执行，即时反馈
+ 脚本式：编写 `.py` 文件，执行 `python filename.py`

#### 2.执行原理
+ 源码(.py) → 字节码(.pyc) → Python虚拟机(PVM) → 运行结果
+ Python 是==解释型语言==（严格说是"先编译后解释"）
+ 字节码是平台无关的中间代码

---

### **(二).动态类型特性**（变量即标签，类型随赋值改变）

+ **变量是标签，不是盒子**
+ **变量本身没有类型，对象才有类型**,无需声明类型，运行时动态确定
+ 赋值只是让变量"指向"对象,同一变量可先后指向**不同类型**的对象
+ 多个变量可以指向同一个对象：a = b = [1, 2, 3]

```python
a = 10      # a 标签贴在 10 这个对象上
a = "hello" # a 标签撕下来，贴到 "hello" 对象上
```

---

### **(三).一切皆对象**

#### 1.什么是"对象"
+ 对象 = 数据（属性）+ 行为（方法）
+ 对象存储在堆内存中，由垃圾回收器管理

#### 2.所有 Python 元素都是对象
+ 可以动态给对象添加属性
+ 可以将函数作为参数传递（高阶函数）
+ 所有类型都有统一的类型体系

#### **1.字面量与c++对比**

* 字面量是直接书写的固定值
* type()语句可得到数据类型，`type(要查看的数据类型)`
* isinstance()检查是否属于指定的类型，返回bool值，`isinstance(数据，类型)`

| 类别 | Python | C++ | 说明 |
|------|--------|-----|------|
| 整数精度 | **无限精度，无溢出** | 受类型范围限制 | Python可计算超大整数 |
| 数字分隔符 | 支持 `1_000_000` | 不支持 | 提高可读性 |
| 布尔值 | `True` / `False`（首字母大写） | `true` / `false` | Python中布尔是int子类 |
| 布尔运算 | `True + True == 2` | 不允许 | **Python布尔可参与算术** |
| 多行字符串 | **三引号** `"""..."""` | 需用 `\n` 或宏 | Python原生支持 |
| 原始字符串 | `r"C:\path"` | 无 | **原始字符串(r)**转义字符失效，按原样输出 |
| 复数 | `3 + 4j` | 无内置 | Python原生支持复数 |
| 空值 | `None`（对象） | `nullptr` / `NULL`（指针） | None是单例，用 `is` 比较 |
| 类型后缀 | 不需要 | `100L` / `3.14f` | Python自动适配 |

#### **2.变量与c++对比**

* 变量是用来存储单个数据的容器
* 一条语句可以定义多个变量，也可以连续赋值(`a,b = 1,"Python"`)

| 对比项 | Python | C++ |
|--------|--------|-----|
| 变量本质 | 标签（贴在对象上） | 内存容器（存储值） |
| 类型绑定 | 变量无类型，对象有类型 | 变量有固定类型 |
| 类型改变 | 允许，随时可指向不同类型 | 不允许 |
| 声明要求 | 不需要，直接赋值即创建 | 必须声明类型 |
| 未初始化 | **不存在（赋值才有变量）** | 存在（值不确定） |
| 指针/引用 | **所有变量都是引用语义** | 分值语义、指针、引用 |
| 内存管理 | 自动垃圾回收 | 手动 new/delete |

## **五、流程控制**

### **(一).缩进规则（无大括号）**

#### 1.缩进（Python vs C++ 核心差异）

| 对比项 | Python | C++ |
|--------|--------|-----|
| 代码块标记 | **缩进**（强制） | 大括号 `{}`（缩进仅用于可读性） |
| 缩进量 | 统一（通常4空格） | 无要求 |
| 混合缩进 | ❌ 禁止（Tab/空格混用报错） | ✅ 允许 |
| 代码块结束 | 缩进减少即结束 | 右大括号 `}` |

#### 2. 核心原理

**Python：缩进即语法**
+ 同一代码块必须缩进一致
+ 嵌套层级增加缩进
+ 建议使用4空格（PEP 8标准）

```python
# 缩进决定代码归属
if True:
    print("属于if")      # 4空格缩进
    print("也属于if")
print("不属于if")         # 缩进恢复，if块结束
```

### **(三).match-case**（模式匹配，Python 3.10+）

| 对比项 | Python `match-case` | C++ `switch` |
|--------|---------------------|--------------|
| 语法 | `match 值:`<br>`    case 模式:`<br>`        代码`<br>`    case _:`<br>`        默认代码` | `switch (表达式) {`<br>`    case 常量:`<br>`        代码; break;`<br>`    default:`<br>`        默认代码; break;`<br>`}` |
| 匹配值类型 | **任意类型** | 整型、枚举、字符 |
| 字符串匹配 | ✅ 支持 | ❌ 不支持 |
| 类型匹配 | ✅ 支持 | ❌ 不支持 |
| 解包匹配 | ✅ 支持（列表、元组） | ❌ 不支持 |
| 条件守卫（guard） | ✅ 支持 | ❌ 不支持 |
| **多个值匹配** | `\|` 分隔 | 连续写多个 `case` |
| `break` 需要 | ❌ **不需要（不会贯穿）** | ✅ 需要 |
| 默认分支 | `case _:`(通配符，匹配任意值) | `default:` |

* `case x:` 会将匹配到的值**绑定到变量 `x`**，后续代码可以使用该变量。

```python
value = 42
# 使用 case _（不绑定）
match value:
    case 1:
        print("匹配到1")
    case _:
        print("匹配到其他值")   # 无法知道具体是什么值

# 使用 case x（变量绑定）    变量名可以任意取
match value:
    case 1:
        print("匹配到1")
    case other:
        print(f"匹配到: {other}")   # 可以使用 other，输出 42
```

#### 2.列表（list）操作

##### (1).增删改查

| 操作 | 方法/语法 | 说明 | 示例 |
|------|----------|------|------|
| **增** | `lst.append(x)` | 末尾添加 | `[1,2] → [1,2,x]` |
| | `lst.insert(i, x)` | 指定位置插入 | `[1,3] → insert(1,2) → [1,2,3]` |
| | `lst.extend(iter)` | 合并列表 | `[1,2] → extend([3,4]) → [1,2,3,4]` |
| **删** | `lst.pop()` | 移除并返回最后一个 | `[1,2,3] → pop() → 3, lst=[1,2]` |
| | `lst.pop(i)` | 移除并返回**索引** i | `[1,2,3] → pop(1) → 2, lst=[1,3]` |
| | `lst.remove(x)` | 移除第一个值为 x 的元素 | `[1,2,1] → remove(1) → [2,1]` |
| | `lst.clear()` | 清空所有元素 | `[1,2,3] → clear() → []` |
| | `del lst[i]` | 删除索引 i 的元素 | `[1,2,3] → del lst[1] → [1,3]` |
| **改** | `lst[i] = x` | 直接赋值修改 | `[1,2] → lst[1]=10 → [1,10]` |
| **查** | `lst[i]` | 访问索引 i 的元素 | `[1,2,3][1] → 2` |
| | `lst.index(x)` | 返回第一个值为 x 的索引 | `[1,2,3].index(2) → 1` |
| | `x in lst` | 判断是否存在 | `2 in [1,2,3] → True` |
| | `lst.count(x)` | 统计 x 出现次数 | `[1,2,1].count(1) → 2` |

##### (2).排序与反转

| 操作 | 方法/函数 | 说明 | 示例 |
|------|----------|------|------|
| 排序 | `lst.sort()` | 原地升序排序 | `[3,1,2] → sort() → [1,2,3]` |
| 降序 | `lst.sort(reverse=True)` | 原地降序排序 | `[1,2,3] → sort(reverse=True) → [3,2,1]` |
| 反转 | `lst.reverse()` | 原地反转 | `[1,2,3] → reverse() → [3,2,1]` |
| 排序（新） | `sorted(lst)` | 返回新列表，原列表不变 | `sorted([3,1,2]) → [1,2,3]` |

##### (3).复制

| 方法 | 说明 |
|------|------|
| `lst2 = lst.copy()` | 浅拷贝 |
| `lst2 = lst[:]` | 切片复制（浅拷贝） |
| `copy.deepcopy(lst)` | 深拷贝（需 `import copy`） |

##### (4).解包操作

| 操作 | 语法 | 示例 |
|------|------|------|
| 基础解包 | `a, b, c = [1,2,3]` | `a=1, b=2, c=3` |
| 变量交换 | `x, y = y, x` | `x=10, y=20 → x=20, y=10` |
| 星号解包 | `first, *rest = [1,2,3,4]` | `first=1, rest=[2,3,4]` |
| 星号解包（尾） | `*first, last = [1,2,3,4]` | `first=[1,2,3], last=4` |

##### (5).列表推导式

| 类型 | 语法 | 示例 |
|------|------|------|
| 基础 | `[表达式 for 变量 in 可迭代对象]` | `[x**2 for x in range(5)] → [0,1,4,9,16]` |
| 带条件 | `[表达式 for 变量 in 可迭代对象 if 条件]` | `[x for x in range(10) if x%2==0] → [0,2,4,6,8]` |
| 带 if-else | `[真值 if 条件 else 假值 for 变量 in 可迭代对象]` | `["偶" if x%2==0 else "奇" for x in range(3)] → ["偶","奇","偶"]` |
| 嵌套 | `[表达式 for 变量1 in 可迭代1 for 变量2 in 可迭代2]` | `[(x,y) for x in range(2) for y in range(2)] → [(0,0),(0,1),(1,0),(1,1)]` |

##### (6).其他常用操作

| 操作 | 语法 | 说明 | 示例 |
|------|------|------|------|
| 拼接 | `lst1 + lst2` | 合并两个列表，返回新列表 | `[1,2] + [3,4] → [1,2,3,4]` |
| 重复 | `lst * n` | 列表重复 n 次，返回新列表 | `[1,2] * 3 → [1,2,1,2,1,2]` |
| 长度 | `len(lst)` | 返回列表元素个数 | `len([1,2,3]) → 3` |
| 遍历元素 | `for item in lst:` | 遍历每个元素 | 输出 `1,2,3` |
| 遍历索引+元素 | `for i, item in enumerate(lst):` | 同时获取索引和值 | `0:1, 1:2, 2:3` |
