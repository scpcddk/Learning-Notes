# 1.Random
+ 导入模块:`import random`

| 函数 | 描述 |
|------|------|
| `random.random()` | 返回 [0.0, 1.0) 范围内的随机浮点数 |
| `random.uniform(a, b)` | 返回 [a, b] 或 [a, b) 范围内的随机**浮点数** |
| `random.randint(a, b)` | 返回 [a, b] 范围内的随机**整数**（包含两端） |
| `random.randrange(start, stop[, step])` | 返回范围内随机整数，与 range() 参数类似，不包含 stop |
| `random.choice(seq)` | 从非空序列中随机选择一个元素 |
| `random.choices(population, weights=None, k=1)` | 从序列中随机选择 k 个元素（可重复） |
| `random.sample(population, k)` | 从序列中随机选择 k 个不重复元素 |
| `random.shuffle(x)` | 就地打乱列表顺序 |
