# 冒泡排序

## 一.核心思想
==**通过相邻元素的两两比较与交换，将较大元素逐步“冒泡”序列末尾。**==

---

## 二.算法原理
+ **遍历比较**：从序列开头开始，**依次比较相邻的两个元素**。
+ **交换位置**：如果前一个元素大于后一个元素，则交换两者的位置。
+ 重复过程：每一轮遍历都会将当前未排序部分的最大元素“冒泡”到末尾；重复多轮，直到所有元素有序

---

## 三.==代码==
```
#include <stdio.h>
// 冒泡排序函数
void bubbleSort(int arr[], int n) 
{
    for (int i = 0; i < n - 1; i++) 
    {
        // 标记本轮是否发生交换，优化有序序列的情况
        int swapped = 0;
        // 每轮遍历后，末尾i个元素已有序，无需再比较
        for (int j = 0; j < n - i - 1; j++) 
        {
            if (arr[j] > arr[j + 1]) 
            {
                // 交换arr[j]和arr[j+1]
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                swapped = 1;
            }
        }
        // 如果本轮没有发生交换，说明序列已有序，提前退出
        if (!swapped) 
        {
            break;
        }
    }
}
```

---

## 四.算法特性

- **时间复杂度**：
  - 最坏/平均情况：O(n^2)（序列完全逆序时，需进行n-1轮遍历，每轮比较n-i-1次）。
  - 最好情况：O(n)（序列已有序时，通过 swapped 标记可提前退出）。
- **空间复杂度**：O(1)（仅使用常数额外空间，属于原地排序）。
- **稳定性**：稳定（相等元素不会交换位置）。

---

## 五.==适用场景==

冒泡排序逻辑简单，但效率较低，仅适用于**数据量较小或接近有序的序列**；实际开发中，更常用高效的排序算法（如快速排序、归并排序）。
