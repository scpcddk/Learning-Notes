# 插入排序

## 一.**核心思想**
==将序列分为已排序区域和未排序区域，每次从“未排序区域”取出第一个元素，插入到“已排序区域”的合适位置，使已排序区域保持有序，逐步扩大已排序区域直至整个序列有序。==

---

## 二.算法原理
1. 初始状态：已排序区域只有序列的第一个元素，未排序区域是剩余所有元素。
2. 第i轮（i从1开始）：
- 取出未排序区域的第一个元素（下标为 i ），暂存为 temp 。
- 从已排序区域的末尾（下标为 i-1 ）向前遍历，**若当前元素大于 temp ，则将该元素后移一位**。
- **找到第一个小于等于 temp 的元素，将 temp 插入到该元素的下一个位置**。
3. 重复步骤：直到未排序区域为空，排序完成。

---

## 三.==**代码**==

```
void insertionSort(int arr[], int n)
{
    for (int i = 1; i < n; i++)
    {
        int temp = arr[i]; // 取出未排序区域的第一个元素
        int j = i - 1;     // 已排序区域的末尾下标
        // 在已排序区域中找到插入位置并移动元素
        while (j >= 0 && arr[j] > temp)
        {
            arr[j + 1] = arr[j]; // 元素后移，腾出插入位置
            j--;
        }
        arr[j + 1] = temp; // 将元素插入到正确位置
    }
}
```

---

## 四.算法特性
- **时间复杂度**：
  - 最好情况（序列已有序）：O(n)
  - 最坏情况（序列逆序）：O(n^2)
  - 平均情况：O(n^2)
- **空间复杂度**：O(1)（原地排序，仅用常数额外空间）
- **稳定性**：稳定（相等元素的相对顺序不会改变）

---

## 五.==适用场景==
1. 数据量较小或部分有序的序列。
2. 对排序稳定性有要求的场景。
3. 在线排序（可边接收数据边排序）。
