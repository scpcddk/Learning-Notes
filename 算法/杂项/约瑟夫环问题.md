# 约瑟夫环问题
## 一.问题描述
n个人围成一圈，从1开始报数，报到m的人淘汰，直到剩下最后1人，求该人的编号。
## 二.核心思路
1. 用数组模拟人群，1~n 为初始编号
2. 淘汰的人标记为 0，不再参与报数
3. 循环遍历数组，报数到m则淘汰，直至剩1人
4. 遍历数组找到未被标记（非0）的数，即为答案
## 三.代码
#include <stdio.h>
int main() 
{
    int n, m;                       // 输入人数n、报数m
    scanf("%d %d", &n, &m);         // 初始化数组，存储1~n编号
    int a[101] = {0};
    for (int i = 0; i < n; i++)
    a[i] = i + 1;
    int remain = n;   // 剩余人数
    int count = 0;    // 当前报数
    int idx = 0;      // 遍历下标b
    while (remain > 1)             // 只剩1人时停止
    {
        if (a[idx] != 0)           // 未淘汰的人参与报数
        count++;
        if (count == m)            // 报到m，标记淘汰
        {
            a[idx] = 0;
            remain--;
            count = 0;
        }
        idx = (idx + 1) % n;       // 循环遍历，模拟围成圈
    }
    for (int i = 0; i < n; i++)    // 输出最后剩余的人
    {
        if (a[i] != 0) 
        {
            printf("%d\n", a[i]);
            break;
        }
    }
    return 0;
}
## 四.变量说明
| 变量 | 含义 |
| ---- | ---- |
| n | 总人数 |
| m | 报数到m淘汰 |
| a[101] | 存储人员编号，0表示已淘汰 |
| remain | 剩余未淘汰人数 |
| count | 当前报数计数 |
| idx | 数组遍历下标 |
