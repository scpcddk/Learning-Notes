# 埃拉托斯特尼筛法（Sieve of Eratosthenes）
## 一.核心思想
==“标记素数的倍数为非素数，剩余未被标记的即为素数”。==
## 二.算法步骤
1. 初始化标记数组：创建一个长度为  N+1  的布尔数组  isPrime ，初始时所有元素设为  True （表示“假设为素数”）。
2. 排除非素数：
- 先将  isPrime[0]  和  isPrime[1]  设为  False （因为 0 和 1 不是素数）。
- 从  i=2  开始遍历到  \sqrt{N} ：
- 若  isPrime[i]  为  True （说明  i  是素数），则将  i  的所有倍数（从  i \times i  开始，因为更小的倍数已被更小的素数标记过）标记为  False （非素数）。
3. 收集素数：遍历  isPrime  数组，所有值为  True  的下标即为素数。
## 三.时间复杂度
埃拉托斯特尼筛法的时间复杂度为  O(n \log \log n) ，是筛选素数的高效算法（远优于“试除法”的  O(n\sqrt{n})  复杂度）。
## 四.代码
#include <stdio.h>
#include <math.h>
int main() 
{
    long long N;
    scanf("%lld", &N);
    int isPrime[N + 1];
    // 初始化：假设所有数都是素数
    for (long long i = 0; i <= N; i++) 
    isPrime[i] = 1;
    // 0和1不是素数
    isPrime[0] = isPrime[1] = 0;
    // 筛法核心逻辑
    for (long long i = 2; i * i <= N; i++) 
    {
        if (isPrime[i]) 
        {
            for (long long j = i * i; j <= N; j += i)
            isPrime[j] = 0;
        }
    }
    // 输出素数
    for (long long i = 2; i <= N; i++) 
    {
        if (isPrime[i])
        printf("%lld ", i);
    }
    return 0;
}
该算法的优势在于一次遍历即可标记所有非素数，特别适合处理  N  较大的场景（如题目中  N \leq 10^5  的范围）。
